# Public Holidays between the years 1990 and 2030

A dataset containing the public holidays for 100 Countries.

## Usage

``` r
NagerHolidayData
```

## Format

A list containing:

- NagerHolidayData:

  a data.frame with the specifics of the Holiday dates by country and
  year

- HolidayVectorperCountry:

  a list of vectors containing the public holiday dates per country

- HolidayDataFrame:

  a dataframe with two columns date (of the holiday) and the iso3c
  countryindication of the country where the holiday is located

## Source
