# Show the available methods of a Refinitiv connection object

Show the available methods of a Refinitiv connection object

## Usage

``` r
EikonShowAttributes(EikonObject)
```

## Arguments

- EikonObject:

  Connection object generated by EikonConnect or RefinitivJsonConnect

## Value

a character vector of attribute names

## Examples

``` r
if (FALSE) { # \dontrun{
Eikon <- EikonConnect(Eikonapplication_id = "your key")
EikonShowAttributes(EikonObject = Eikon)
} # }
```
