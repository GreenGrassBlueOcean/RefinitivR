# Install Eikon Python environment (Defunct)

\`r lifecycle::badge("defunct")\`

Python/reticulate support has been removed from RefinitivR. The package
now uses a direct JSON connection via httr2. Use RefinitivJsonConnect()
or RDConnect() to connect.

## Usage

``` r
install_eikon(...)
```

## Arguments

- ...:

  All arguments are ignored.

## Value

This function always raises an error.
