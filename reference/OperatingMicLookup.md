# Operating Mic lookup from Reuters Exchange Code

A data.frame containing a lookup list from Reuters exchange codes to
operating mics

## Usage

``` r
OperatingMicLookup
```

## Format

A data.frame containing:

- RDN_EXCHD2:

  Reuters Exchange Code

- repairedMIC:

  An operating mic for this Reuters exchange Code

## Source

derived from Reuters Eikon terminal
