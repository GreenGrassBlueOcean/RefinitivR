# Flatten a nested list put list format in data.table format way (don't mix up lists and vectors in one nested list)

Flatten a nested list put list format in data.table format way (don't
mix up lists and vectors in one nested list)

## Usage

``` r
flattenNestedlist(data)
```

## Arguments

- data:

  a nested list

## Value

a list

## Examples

``` r
# Refinitiv:::flattenNestedlist(list(list("a", "b"), c(1,2)))
```
