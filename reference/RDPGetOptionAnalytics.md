# Get RDP option analytics

\`r lifecycle::badge("defunct")\`

This function required a Python connection and has been removed. Use the
official LSEG Data Library for Python (lseg-data) directly for option
analytics.

## Usage

``` r
RDPGetOptionAnalytics(...)
```

## Arguments

- ...:

  All arguments are ignored.

## Value

This function always raises an error.
