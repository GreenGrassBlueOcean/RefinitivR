# Lookup code for SearchViews to alternate between RD and JSON format SearchViews

A data.table containing a lookup list from RD SearchViews to JSON
SearchViews

## Usage

``` r
SearchViewsLookup
```

## Format

A data.table containing:

- SearchViews_RD:

  SearchViews valid for the RD API

- SearchViews_JSON_RDP:

  SearchViews valid for the RDP API and direct JSON calls
